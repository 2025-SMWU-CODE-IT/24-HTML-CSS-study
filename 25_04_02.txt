[스터디내용]
: HTML의 기본적인 기능들에게 대해서 학습함.


[세부내용]

#1 <strong>내용 </strong> : 글자 강조

#2 <a href=(링크) target="_blank”>내용</a>
href - 하이퍼링크

공백으로 두 속성 구분

target="_blank” - 하이퍼링크 누르면 새창에서 열림

#3 <li>html</li> : 리스트 사용

각각의 그룹으로 나누어 표현하는 법
<ul></ul>, <ol></ol>
→ul =unordered list
→ol =ordered list(숫자가 붗어서 표현됨)

#4 <body>의 의미
본문내용은 바디 안에 있어야함

#5 <head>
문서의 정보가 헤드 태그안에 들어감
<title>을 헤드태그안에 사용하면 문서의 제목으로 사용할 수 있음

#6 <html>
헤드와 바디 위의 상위 태그로 html문서라는 것을 브라우저에 알려주는 것
